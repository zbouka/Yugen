# Yugen

An android app made using Flutter and Firebase,and it can be used to watch anime/manga, powered by additional features.

## Features
- Intuitive and attractive user interface.
- Extensive library of animes and mangas.
- High quality streaming playback.
- Download option to view content offline.
- Personalization of the user experience.
- Get the latest news about anything related to animes/mangas
- Browse and apply a variety of wallpapers optimized for both phones and tablets
- Share your thoughts by using the in-app comment sections
